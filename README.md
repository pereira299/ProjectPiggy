# PigGrow

## Pacotes utilizados
    - Lucide        -> Icones
    - Next          -> Framework React, oferece otimizações e recursos adicionais
    - React         -> Framework para interfaces reativas
    - Tailwindcss   -> Estilização
    - Material UI   -> Biblioteca de componentes

## Estrutura de pastas
|- node_modules     -> todos os pacotes utilizados
|- public           -> todos os recursos publicos para o usuario (imagens, videos, ...) 
|- src              
    |- app          -> Paginas do site 
    |- components   -> Componentes utilizados pelas paginas, e por outros componentes
    |- services     -> Integração com serviços externos
    |- types        -> Tipagem
    |- utils        -> Funções utilizadas em diferentes partes do sistema


## Comandos
    - ```yarn install``` Instalar dependencias do projeto
    - ```yarn dev```  Roda o sistema em modo de desenvolvimento
    - ```yarn build``` Prepara o sistema para produção
    - ```yarn start``` Roda o sistema em produção
